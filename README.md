## ✨ 功能

- **mpv 驱动** — 全格式支持，解码能力强
- **macOS 风格** — 简洁优雅，无边框设计
- **Windows 专属** — DWM Acrylic 模糊透明效果
- **三栏布局** — 媒体库 / 时间轴 / 独立播放窗口
- **自动播放** — 按边双击调整窗口大小

## 📦 安装

```bash
pip install -r requirements.txt
```

## 🚀 运行

```bash
python main.py
```
